# A/B Test Conversion Rate Chart

Interactive line chart to visualize A/B test conversion rates over time.

## Tech stack

- React + TypeScript
- Vite
- Recharts
- CSS Modules

## Features

- Conversion rate (%) line chart for all variations
- Day / Week aggregation
- Variation selector (at least one variation always selected)
- Adaptive X/Y axes based on visible series
- Hover tooltip with vertical guideline and percent values
- Responsive layout (671â€“1300px)

## Bonus features

- Line style selector:
  - Straight linear
  - Curve cardinal
  - Area
  - Shadow
- Light / Dark theme toggle
